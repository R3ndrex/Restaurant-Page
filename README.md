# Restaurant-Page

This project is a single page web application that implements dynamic navigation without page reload. I use Webpack to build JavaScript modules, each of which is responsible for displaying a separate section of the site (menu, about, main). When you click on navigation links, JavaScript changes the content of the page, creating a transition effect between sections.
